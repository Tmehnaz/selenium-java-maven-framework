# Selenium Java Automation Framework

## Tech Stack
- Java 17
- Selenium 4.43.0
- TestNG 7.12.0
- Maven 3.9.15
- WebDriverManager 6.3.3
- Page Object Model (POM)

## Project Structure
- pages/ - Page Object classes
- tests/ - Test classes
- utils/ - Base test and helper classes

## How to Run
mvn clean test

## Design Patterns Used
- Page Object Model (POM)
- Data Driven via config.properties

## Test Cases
- Google Search functionality test

## CI/CD
Automated with GitHub Actions - runs Selenium tests on every push
